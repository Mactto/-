# 로프

---
[ Greedy Algorithm ]

![change](./images/로프.PNG)

```python
def cal_change(change):
    changeNum = 0
    while change != 0:
        for changeMoney in [500,100,50,10,5,1]:
            changeNum += change // changeMoney
            change = change % changeMoney
    print(changeNum)


if __name__ == "__main__":
    price = int(input())
    cal_change(1000-price)
```
