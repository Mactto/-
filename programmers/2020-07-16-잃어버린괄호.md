# 잃어버린 괄호

![image](./images/잃어버린괄호.PNG)

```python
def cal(s):
    num = []
    for n in s:
        sum_n = 0
        s_num = n.split('+')
        for r in s_num:
            sum_n += int(r)
        num.append(sum_n)
    result = num[0]
    for a in range(1, len(num)):
        result -= num[a]
    print(result)
if __name__ == "__main__":
    s = input().split('-')
    cal(s)
```
